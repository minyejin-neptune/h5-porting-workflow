# 포팅 기준 체크리스트 (정답지)

> **이 문서가 기준이다.** H5 포팅 워크플로우(스킬·에이전트·검증 스크립트)는 이 체크리스트의 모든 항목을 담아야 한다.
> 워크플로우를 수정·검증할 때 이 문서와 대조한다 (커버리지 갭 = 워크플로우 결함).
> 출처: 사람이 수동 포팅하며 확립한 체크 표 (2026-07-06 repo 등록). **항목 추가·수정은 사용자 승인으로만.**
